# Changelog

## [0.0.1] - 2025-07-12

### Added
- 初始版本发布
- 核心PDF生成框架
- 模块化设计架构
- 标题模块 (TitleModule)
- 主表模块 (MainTableModule) 
- 子表模块 (SubTableModule)
- 数据适配器系统
- 配置管理系统
- 样式主题支持
- 示例应用

### Features
- 支持多种PDF模块
- 高度可配置的样式系统
- 灵活的数据适配机制
- 自动分页处理
- 跨列布局支持
- 自定义主题

### Technical
- 基于 pdf ^3.11.3
- 支持 Flutter 3.0+
- 符合 Dart 包发布规范
- 完整的类型安全
